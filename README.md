# Lista de Tarefas

Este projeto foi desenvolvido para criar uma lista de tarefas utilizando **HTML**, **CSS** e **JavaScript**. A ideia principal do projeto é permitir que os usuários adicionem, removam e marquem tarefas como concluídas de maneira simples e interativa.

## Tecnologias utilizadas

- **HTML**: Estrutura básica da página e formulário para inserir tarefas.
- **CSS**: Estilos para tornar a aplicação visualmente agradável e responsiva.
- **JavaScript**: Lógica para gerenciar a lista de tarefas, incluindo adicionar, remover e marcar como concluída.

## Funcionalidades

- Adicionar novas tarefas.
- Marcar tarefas como concluídas.
- Remover tarefas da lista.
- As tarefas são salvas localmente no navegador (utilizando `localStorage`), garantindo que a lista seja preservada mesmo após o fechamento do navegador.

## Como executar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` no seu navegador para visualizar a aplicação funcionando.
